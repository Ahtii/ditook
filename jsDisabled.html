Please enable the javaScript and refresh the page.

Below are the steps to enable javaScript in some of the common browsers:

Google Chrome

  • Goto settings.
  • Advanced.
  • Under Privacy and security section click on Content Settings.
  • Click on JavaScript.
  • Click on Blocked slider.

Opera

  • Click on opera logo.
  • Goto settings.
  • Click on Websites section.
  • Under JavaScript Heading Select   Allow all sites to run JavaScript .

Mozilla

  • Type about:config in the address bar.
  • A warning message will be displayed, Click on the button labeled I'll be careful, I promise! or I accept the risk! .
  • In the search bar type javascript.enabled .
  • Double click on the search result to enable/disable the javascript.

Safari

  • Goto Settings.
  • Click on Preferences.
  • Select Security tab. .
  • Check the Enable JavaScript option.
